## Funcionalidades

- Cadastro
- Listagem 
- Alteração
- Exclusão
- Adiciona um cpf  quando a uf é selecionada como "SP" , no cadastro
- Imprede registro de cliente com idade menor que 18 anos , quando a uf é selecionada como "MG" , no cadastro
